# new-macaukaijiang-guide

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为通用页面存档与展示用途。

## 项目简介

本项目旨在集中存放一批独立的 HTML 页面，这些页面可能包含信息展示、工具界面或静态内容。所有页面均为独立文件，不依赖后端服务，可直接通过浏览器或 GitHub Pages 进行访问。

## 目录说明

```
.
├── index.html          # 入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：可选的根页面，用于导航或说明。
- `pages/`：所有归档的 HTML 页面均放置于此目录下，每个页面独立完整。
- `assets/`：存放页面共同依赖的样式、脚本或图片资源。

## 页面归档说明

- 每个 HTML 页面均为自包含文件（可引用 `assets/` 下的公共资源）。
- 页面内容不指向任何具体第三方网站、服务或商业推广。
- 归档仅用于技术演示、个人学习或静态内容展示。
- 所有页面均不包含恶意代码、跟踪脚本或诱导性内容。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面。
- 欢迎通过 Issue 提交建议或报告问题，但请注意不要提交具体域名或商业链接。
- 所有页面更新会保留历史提交记录，便于回溯。
- 如需自行部署，可直接克隆仓库并通过 GitHub Pages 或其他静态托管服务发布。

本仓库仅作为 HTML 页面归档与展示用途。页面内容由维护者提供，不构成任何形式的外部推荐或承诺。使用者应自行判断内容的适用性。
